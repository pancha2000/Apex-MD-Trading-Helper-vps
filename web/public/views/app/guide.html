📱 WhatsApp Integration

Linking WhatsApp lets you receive trading signals, TP/SL hit notifications, and daily P&L reports directly.

1 Go to Settings → Click Generate Token
2 Open WhatsApp → Send this message to bot number +94 70 139 1585:
.link LINK-XXXXXX
Replace XXXXXX with your token code
3 Bot replies with ✅ confirmation — you're linked!

To unlink: Send .unlink | Check status: .linkstatus

⚡ AI Scanner

The scanner uses 70+ technical indicators to find high-probability trading setups.

  • Deep Scan — Enter any coin (eg: BTC, ETH, SOL) + timeframe → full analysis
  • Auto Market Scan — Scans top 50 coins automatically → shows top 5 setups
  • Score — 0-100. Higher = stronger signal. Your threshold filters out weak signals
  • Funding Rate — High positive = LONG squeeze risk. High negative = SHORT squeeze risk
  • Daily Trend — ✅ = trading with daily trend. ⚠️ = against daily (higher risk)
  • RRR — Risk:Reward Ratio. Minimum 1.5:1 required for quality setups
💡 Tip: Set your Score Threshold to 40+ in Settings for better quality signals.

📄 Paper Trading

Practice trading with virtual money — zero real risk, full realism.

  • Start with $100 virtual balance (customizable)
  • Open trades manually or directly from Scanner results
  • TP1 auto-calculated as midpoint between Entry and TP2
  • Live P&L updates every 30 seconds
  • Max 5 open positions at once
  • Trades auto-deleted after 7 days

🔔 Price Alerts

Set price level notifications for any coin. Delivered via WhatsApp when triggered.

  • Up to 20 simultaneous alerts
  • Set "above" or "below" target price conditions
  • WhatsApp must be linked to receive notifications

⚙️ Settings Guide

  • Score Threshold — Minimum signal score for Scanner. Recommended: 40
  • Capital — Set your paper trading balance
  • Trading Mode — Signals Only: receive alerts. Auto Trade: bot executes (requires API keys)
  • API Keys — Add Binance API key (secret optional for read-only). Never share secret keys
  • AI Keys — Add your own GROQ key for AI Chat. Free at console.groq.com
